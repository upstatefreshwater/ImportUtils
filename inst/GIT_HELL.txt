# To pull in updates to the remote main:

1) Check GIT status - look to see if there are locally modified files
	git status
	
1) Check if anything changed remotely
	git fetch
	
“Show me what commits exist on origin/main that I don’t yet have on main.”
	git log --oneline --decorate --graph --boundary main origin/main
	
2) If there are modified files from status check
	git commit -am "WIP: save before syncing main"
	or
	git stash
	
3) Switch to main, update it
	git switch main
	git pull -ff-only origin main
	
4) Switch back to working branch
	git switch <branch-name>
	
5) Rebase (from the branch!) NOTE!!! - This assumes private branches, if branches are shared use a merge
	git rebase main 
	
6) If you stashed, bring back the work
	git stash pop
